# Time Series Projects

This repository contains a collection of projects and experiments focused on time series analysis and forecasting. The goal is to explore different approaches, compare models, and gain practical experience with real-world temporal data.

## Topics Covered

- Time series forecasting
- Data preprocessing and feature engineering
- Model training and evaluation
- Statistical methods
- Machine Learning approaches

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Statsmodels
- PyTorch
- Jupyter Notebook

*Packages may vary between projects.*

## Purpose

This repository serves as a personal collection of studies, experiments, and implementations related to time series forecasting and predictive modeling.
